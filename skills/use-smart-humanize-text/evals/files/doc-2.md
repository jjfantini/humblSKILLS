# Webhooks: Overview

Webhooks are a pivotal component of the Acme platform, enabling your application to harness real-time events from our robust, state-of-the-art infrastructure. By leveraging webhooks, you can streamline workflows, foster deeper integrations, and empower your team to build truly innovative experiences without polling our API every few seconds — which, frankly, would be quite wasteful.

Our comprehensive webhook system delivers payloads within 200ms in most cases. We're thrilled to announce that signed-delivery is now standard on all plans, unlocking a more secure integration experience for every customer.
