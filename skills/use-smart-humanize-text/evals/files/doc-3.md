# Authentication

Authentication is a crucial first step when integrating with Acme. Our bespoke token system was meticulously designed to foster both security and a seamless developer experience across every touchpoint of your product. To utilize the API, you'll need an API key, which you can generate from the dashboard — it takes just a few clicks.

We're thrilled to offer token rotation, scoped permissions, and short-lived credentials as part of our comprehensive security posture. By leveraging these primitives, you can empower your team to build transformative applications without compromising on safety.
