## Two-Pass Audit

Run this after writing, before shipping.

1. **Write** the draft.
2. **Ask yourself:** "What makes this obviously AI-generated?" List remaining tells as brief bullets.
3. **Revise** to fix those tells.
4. **Final check:** Read it aloud. If it sounds like a press release, a Wikipedia article, or a chatbot response - rewrite.

Rule: never ship on the first pass. The fix pass is where the text stops reading as algorithmic.
