## Title Case in Headings

AI capitalizes all main words. Humans use sentence case except for proper nouns.

**Incorrect:**

```markdown
## Strategic Negotiations And Global Partnerships
```

**Correct:**

```markdown
## Strategic negotiations and global partnerships
```

Rule: only capitalize the first word and proper nouns. This applies to all heading levels (`#`, `##`, `###`).
