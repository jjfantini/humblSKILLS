## Plain Replacements

Replace with plain words.

| Banned | Plain |
|--------|-------|
| crucial | important |
| showcases | shows |
| bolsters | supports |
| boasts | has |
| intricate | complicated |
| enduring | lasting |
| valuable insights | useful |

General principle: if a shorter, simpler word would communicate the same thing, use it. AI reaches for long Latinate words; humans reach for short Germanic ones.
