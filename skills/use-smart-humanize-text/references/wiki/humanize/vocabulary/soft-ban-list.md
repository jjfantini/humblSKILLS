## Soft-Ban Words

Not always wrong, but almost always a plainer alternative exists. Default to the plainer word.

**Find a plainer alternative:**

crucial, enhance/enhancing, highlighting, emphasizing, align with, valuable insights, commitment to, diverse array, ensuring, cultivating, encompassing, resonate with, leveraging, furthermore, additionally (to start a sentence), moreover, notably, transformative, comprehensive (as filler)

See `plain-replacements.md` for canonical substitutes.
