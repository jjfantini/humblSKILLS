## Sycophantic / Servile Tone

AI sounds like a chatbot assistant. Cut flattery; start with substance.

**Incorrect:**

```markdown
Great question! You're absolutely right that this is a complex topic. That's an excellent point.
```

**Correct:**

Cut the flattery. Start with the substance.

```markdown
Access patterns matter more than indexing strategy here.
```

Rule: the first sentence should contain information the reader didn't already have. Compliments and affirmations are information-free.
