## Filler Phrases

Remove these on sight. Each multi-word phrase has a one- or two-word substitute that says the same thing.

| Filler | Plain |
|--------|-------|
| In order to achieve this goal | To achieve this |
| Due to the fact that | Because |
| At this point in time | Now |
| In the event that | If |
| has the ability to | can |
| It is important to note that | (cut entirely, just state the thing) |

Rule: if a phrase is longer than three words and you can replace it with one, do.
